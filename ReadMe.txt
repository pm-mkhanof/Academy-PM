Проект "Академия ФМ" (Academy PM)(далее - проект) - есть бесплатная образовательная платформа на базе образовательной программы физико-математических классов Лицея №39 г. Махачкала.

Проект - есть идея ученика 9 физико-математического класса (на момент учебного года 2023/2024) Магомедханова Магомеда Маратовича.

Проект - есть реализация при поддержке большого количества лучших учеников в разных направлениях 39 лицея, для создания материала были отобраны лучшие из лучших учеников.

Цель проекта - помощь ученикам в освоении нелегкого материала.
